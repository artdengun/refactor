Mulai Analisis Ulasan Pengguna

Gunakan sistem Smart Review Analyzer (SRA) untuk menganalisis ulasan pengguna aplikasi pinjaman online. Anda dapat mengunggah dataset baru (.csv) untuk dianalisis menggunakan model IndoBERT Multitask Learning, atau melihat hasil analisis dan rating dari aplikasi yang telah tersedia.

Unggah Dataset Ulasan

Silakan pilih file CSV berisi ulasan pengguna untuk dilakukan analisis otomatis.

Status Proses Analisis

Proses analisis data dilakukan melalui beberapa tahapan utama untuk mengubah teks mentah menjadi data yang siap diprediksi oleh model. Tahapan tersebut meliputi: Case Folding (mengubah huruf menjadi format standar), Cleansing (menghapus karakter tak relevan), Normalization (menyeragamkan penulisan kata), Tokenizing (memecah kalimat menjadi kata), Stopword Removal (menghapus kata umum yang tidak bermakna penting), dan Stemming (mengembalikan kata ke bentuk dasarnya). Semua proses dijalankan secara berurutan untuk memastikan hasil analisis yang akurat sebelum diproses oleh model IndoBERT Multitask.
